# Практична робота №10 — Pokemon (рефакторинг)

**Мета:** Рефакторинг коду, розбиття на модулі, створення класу `Pokemon`, який створює героя та противника.

## Структура
- `index.html` — точка входу (ES Modules)
- `main.js` — створює персонажів за допомогою `Pokemon.createPair` і містить ігрову логіку
- `pokemon.js` — клас `Pokemon` з методами `takeDamage`, `resetHP`, `renderHP` та статичним методом `createPair`
- `utils.js` — утиліти (`randomInt`)
- `log.js` — простий інтерфейс для виводу логу бою

## Як запустити
1. Відкрий `index.html` у сучасному браузері (підтримка ES Modules).
2. Натискай **Атакувати** для виконання раунду бою.
3. **Скинути** повертає початкові HP та очищує лог.
